    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with GalioPay in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the integration example to your local machine:
      
      ```bash
      git clone https://github.com/GalioPay/galiopay-integration-example.git
      cd galiopay-integration-example
      ```
    </Step>

    <Step title="Install dependencies">
      Install the required packages:
      
      ```bash
      npm install
      ```
    </Step>

    <Step title="Configure environment variables">
      Copy the example environment file and add your GalioPay credentials:
      
      ```bash
      cp .env.example .env.local
      ```
      
      Update `.env.local` with your credentials:
      
      ```env
      GALIOPAY_API_BASE=https://pay.galio.app/api
      GALIOPAY_API_KEY=your_galiopay_api_key
      GALIOPAY_CLIENT_ID=your_galiopay_client_id
      APP_URL=http://localhost:3000
      ```
    </Step>

    <Step title="Start the development server">
      Run the Next.js development server:
      
      ```bash
      npm run dev
      ```
      
      Open [http://localhost:3000](http://localhost:3000) to view the checkout page, or visit [http://localhost:3000/examples](http://localhost:3000/examples) for interactive API examples.
    </Step>
  </Steps>
</div>
